Projektit

Java REST API + MySQL

Ongelma: Tarve tallentaa ja hallita dataa jäsennellysti modernin web-sovelluksen kautta.
Ratkaisu: Kehitin REST-palvelun Javalla ja JPA:lla, joka mahdollistaa tietojen luomisen, lukemisen, päivittämisen ja poistamisen MySQL-tietokannasta.

Tutustu GitHubissa
LeJOS EV3 Viivanseuraaja

Ongelma: Kuinka saada autonominen robotti seuraamaan tarkasti ennalta määrättyä reittiä.
Ratkaisu: Rakensin PID-säätöön perustuvan viivaa seuraavan robotin käyttäen Javaa ja LeJOS-alustaa. Robottia voi ohjata etänä HTTP-pyynnöillä.
